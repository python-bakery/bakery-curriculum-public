<p>Write a function <code>all_cats</code> that consumes a comma-separated string of animals and returns whether all of the animals have <code>"cat"</code> in their name. For example, the string <code>"gerbil,catfish,dog,cat"</code> would return&nbsp;<code>False</code>, but just&nbsp;<code>"cat,catfish"</code> would return <code>True</code>. If the function consumes an empty string, then it returns&nbsp;<span style="background-color: rgb(249, 242, 244); color: rgb(199, 37, 78); font-family: Menlo, Monaco, Consolas, &quot;Courier New&quot;, monospace; font-size: 12.6px;">True</span>. Unit test this function.</p>
